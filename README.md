# Проект с примером простого EFI-приложения

#### Порядок сборки проекта
```bash
make build_submodule
make
```
#### Для записи на загрузочную флешку
- Подготовить флешку:
    - создать на ней раздел `EFI System`
    - создать каталог в этом разделе /EFI/BOOT
    - смонтировать раздел в каталог `/mnt`
    - выполнить в терминале `make copy`
